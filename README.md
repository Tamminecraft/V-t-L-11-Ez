# Vật Lý 11 Ez

Đã tạo lại frontend thành React/Vite với cấu trúc route riêng cho:

- `Trang chủ`
- `Bài giảng`
- `Phòng thí nghiệm`
- `Quiz`
- `Flashcards`
- `AI`
- `Dashboard giáo viên`

Backend Express hiện hỗ trợ API lưu:

- `/api/progress`
- `/api/quiz-scores`
- `/api/flashcards`
- `/api/users`
- `/api/reviews`, `/api/qa`, `/api/ai-chat`

## Chạy dự án

1. Cài dependencies frontend:

```bash
cd frontend
npm install
```

2. Build frontend:

```bash
npm run build
```

3. Chạy backend:

```bash
cd ../vatly11ez-server
npm install
npm start
```

4. Mở `http://localhost:8787` nếu đã build frontend hoặc sử dụng `npm run dev` trong `frontend` để chạy dev server.
